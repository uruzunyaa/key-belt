# Build Guide
こちらは自作キーボードキット Key-Belt のビルドガイドになります。

## はじめに
ファームウェアについては、VIA Custom UI for Vial を用いてカスタマイズする事が出来ます。

本製品は組み立てが必要なキットです。

組立工程の中には、ハンダ付けや細いねじを締める工程があります。
あらかじめ必要な工具（後述）を用意した上で、組立を始めてください。

キット内容物の不足や不明点等ありましたら、以下連絡先のいずれかにご連絡ください。

X: [@uruzunyaa](https://x.com/uruzunyaa)

mail: uruzunyaa@gmail.com

## 組み立て

### 初心者の方へ

取り付ける部品にはミドルプレートの付ける面に印が付いています。

基板のどちらの面にはんだ付けしたら良いか分からない時は、基板上に書かれた部品のマークを確認しましょう。

### ダイオードを取り付ける
まず、ダイオードの足を曲げ46個取り付けます。

取り付ける場所は、裏面に書かれている D1 ~ D46 の場所です。

**ダイオードには向きがあり** 、黒く塗られた向きを示す線が入っています。

この線を、基板の三角の先端と縦棒の部分に合わせます。

はんだ付け出来たら、はみ出ている足をニッパーで切ります。

![diode](https://github.com/uruzunyaa/key-belt/blob/main/image/Diode.jpg)

### リセットスイッチを取り付ける
小さい青のボタンが付いたスイッチです。

リセットスイッチをはんだ付けします。この部品に向きはありません。

場所は 中央上部 の W_Pus のように書かれた場所です。
### スライドスイッチを取り付ける

向きはありません。はんだ付けをして、足を切ります。

スライドスイッチは表側中央の ON OFF と書かれた場所です。

### キースイッチを取り付ける
まず、スイッチプレートのうち、角に当たるキーを何個かスイッチプレートにはめ込みます。

**この際、基板の向きに注意してください。ON OFF と書かれている方が表です**

![Outside＿Plate](https://github.com/uruzunyaa/key-belt/blob/main/image/Outside_Plate.jpg)

そのまま、ミドルプレートに通し、はんだ付けします。
先にいくつかはんだ付けする事で組み立て時の歪みを軽減出来ます。

![Outside＿OK](https://github.com/uruzunyaa/key-belt/blob/main/image/Outside_OK.jpg)

その後、残りのキーも全てはんだ付けしましょう。

### 電池ホルダーを取り付ける
**電池ホルダーの向きに注意してください。** 左が **－** 右が**＋** です。

はんだ付けをしたら、足を切ります。

![Battery](https://github.com/uruzunyaa/key-belt/blob/main/image/Battery.jpg)

### BLE Micro proを取り付ける
コンスルーを画像のように、金の点の部分がBLE側になるような向きで差し込みます。

BLEへコンスルーの取り付けが出来たら、そのまま基盤にも差し込みましょう。

**※12ピンのコンスルーをUSBポート側に寄せて使ってください。**

![Conthrough](https://github.com/uruzunyaa/key-belt/blob/main/image/Conthrough.jpg)

差し込んだら、BLE Micropro側のみはんだ付けします。 (はんだ付けしなくても動作しますが、BLEを外したくなった時、上手く外す事が出来ません。)

![PutBLE](https://github.com/uruzunyaa/key-belt/blob/main/image/PutBLE.jpg)

### 電源投下前確認
ネジを取り付ける前に、必ず動作確認を行いましょう。

まず、電池を入れる前に、部品の向きと電池の種類を確認してください。

向きに注意が必要な部品は以下の2つです。

- ダイオード(黒い部分と四角が一致しているか)
- 電池ホルダー(2つとも左側にバネが付いているか)

これらの部品の向きや種類を間違えた状態で電源を入れてしまうと、部品が故障したり、**最悪の場合発火します。**

## ファームウェアの書き込み
ここからはファームウェアの書き込みを行います。

### BLE Micro proの設定
まず、BLE Micropro本体を、Key-Beltであると認識させるための設定を行います。

BLEと有線接続をした状態で、以下サイトにアクセスしてください。

https://sekigon-gonnoc.github.io/BLE-Micro-Pro-WebConfigurator/

### ブートローダーのアップデート
Update Bootloaderのタブをクリックします。

ble_micro_pro_bootloader_1_4_0を選択し、アップデートを行います。

### アプリケーションアップデート
Update Applicationのタブへ移動します。

ble_micro_pro_vial_1_4_0を選択し、アップデートします。

### キーボードの設定
Edit configのタブへ移動します。

Select keyboardの中から、 ble_micro_pro を選択し、アップデートを行います。

### 設定を書き込む
以下のファイルをダウンロードし、BLEMicropro のストレージ直下にアップロードしてください。

https://github.com/uruzunyaa/key-belt/blob/main/key_belt_config.bin

アップロード完了後、USB接続を一度解除し、再度接続する事で、設定が完了します。

### キーマップの設定
キーマップの設定を行います。以下サイトにアクセスし、キーボードを選択し接続します。

https://sekigon-gonnoc.github.io/via-custom-ui-for-vial/

UP SETTING から以下のデフォルトキーマップをアップロードするか、Keymapをクリックし、手動で編集を行ってください。

デフォルトキーマップは以下のファイルをアップロードする事で設定可能です。

https://github.com/uruzunyaa/key-belt/blob/main/key_belt_vial_default.json

### 動作確認
USB接続を解除し、ペアリングを行います。

電源をONにした状態にすると、PC側からBluetoothの欄にKey-beltが出てくるはずです。

![ConnectBLE](https://github.com/uruzunyaa/key-belt/blob/main/image/BMP.png)

接続後、正常に動作するか確認してください。

## ネジの取り付け

### スペーサーを止める
まず、ボトムプレートの枠線が付いている穴にネジ頭を、逆側にスペーサーを止めます。

この時、下部の8個の穴は6mm丸型、上部の左右の穴8個には12mm六角のスペーサーを止めます。

![PutSpacer](https://github.com/uruzunyaa/key-belt/blob/main/image/PutSpacer.jpg)

### ボトムプレートを閉じる

その後、安定プレートは **付けずに** ボトムプレートをねじ止めします。

![BottomPlate](https://github.com/uruzunyaa/key-belt/blob/main/image/BottomPlate.jpg)

### 表側を閉じ、電池プレートを取り付ける
※自室のみで使う等、電池がむき出しで問題ない場合、この工程はスキップしても良いです。筆者は屋外に持ち出す事を想定しているため、電池の落下や水滴が飛び散る事を想定して付けています。

電池を入れていなければ入れてから、電池プレートを取り付けます。
(オスメスのネジを電池の横の穴に入れてから、プレートを通し上からネジ止めします。オスメスネジは手で回せると思います。)

![Battery_Plate](https://github.com/uruzunyaa/key-belt/blob/main/image/Battery_Plate.jpg)

### 安定プレートを付ける

下部にベルトを通し、その上から安定プレートを被せ、ねじ止めします。

画像上部にある横のプレートはねじ止めをするだけで大丈夫です。

![Belt](https://github.com/uruzunyaa/key-belt/blob/main/image/Belt.jpg)
